# haladoprog

A program egy asztali, Tkinter alapú képannotáló alkalmazás, amellyel képeket lehet megnyitni, megjeleníteni és azokon különböző objektumokat megjelölni. A felhasználó a bal oldali vásznon (Canvas) tölthet be képfájlokat, amelyeket a Pillow (PIL) könyvtár segítségével méretez a program úgy, hogy azok beleférjenek egy 450×450 pixeles területbe. Az annotálás kétféle móddal történhet: sokszög (polygon) rajzolásával, ahol egérkattintásokkal pontokat vesz fel a felhasználó, illetve téglalap (box) rajzolásával, ahol kattintás–húzás–elengedés mozdulattal jelölhető ki egy tartomány. A jobb oldali panelen cimkéket lehet létrehozni, listázni, kiválasztani és törölni, ezekhez a program véletlenszerűen vagy manuálisan választott színeket rendel, hogy az annotációk vizuálisan is jól megkülönböztethetők legyenek. Az alkalmazás SQLite adatbázist használ, amelyben két tábla tárolja a cimkéket (labels) és az egyes képekhez tartozó annotációkat (annotations), így a munka több futás között is tartósan megőrizhető. Mentéskor a program elmenti a kép elérési útvonalát, a kiválasztott cimke azonosítóját, az annotáció típusát, a koordinátákat és a hozzá tartozó színt, mindezt időbélyeggel együtt. A felhasználó visszavonhatja az utolsó rajzolási lépést, illetve törölheti az egész vásznat, ami megkönnyíti a hibák javítását és az újraannotálást. A jobb oldali „Statisztika” és „DB Statisztika” részeken a program megjeleníti az aktuális munkamenetben létrehozott annotációk darabszámát és az adatbázisban tárolt összes annotáció számát, cimkénkénti bontásban is. Az alkalmazás bezárásakor rákérdez, hogy a felhasználó el szeretné-e menteni a még nem mentett annotációkat az adatbázisba, így csökkentve az adatvesztés kockázatát. A program ideális egyszerű képi adatgyűjtésre, gépi tanulási tanítóadatok kézi előállítására, valamint kisebb adatbázis-alapú jelölési projektek kezelésére, mivel a Tkinteres felület könnyen használható, az SQLite pedig külön szerver telepítése nélkül biztosít tartós tárolást.
